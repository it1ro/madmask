# Итерация 10: Страница товара — UX/вёрстка улучшения (2026-04-01)

## Цели

- Убрать зависимость ключевого контента от JS (progressive enhancement).
- Сделать CTA/цену заметнее при скролле на desktop.
- Улучшить ориентирование пользователя на странице товара.
- Привести формат цены к ожидаемому для ₽ (без копеек).

## Что изменили (факт, в коде)

- **Progressive enhancement для правой колонки**:
  - По умолчанию `.product-show-info__inner` теперь **видим** (без JS).
  - Анимация reveal включается только когда на `<html>` есть класс `.js`.
  - Файлы:
    - `app/views/layouts/application.html.erb` — добавлен класс `js` на `<html>` ранним inline-script.
    - `app/assets/tailwind/application.css` — reveal-стили обёрнуты в `.js ...`.

- **Sticky правая колонка на desktop**:
  - Правая колонка (`product-show-info`) стала `sticky` на `lg+` с `top-28` (под шапку).
  - Файл: `app/views/products/show.html.erb`.

- **Хлебные крошки**:
  - Добавлен `nav[aria-label="Хлебные крошки"]`: `Каталог / Категория / Товар`.
  - Категория ведёт на каталог с фильтром `?category=...`.
  - Файл: `app/views/products/show.html.erb`.

- **Формат цены**:
  - На странице товара цена теперь показывается без копеек (`precision: 0`).
  - Файл: `app/views/products/show.html.erb`.

- **CTA на странице товара: корзина/избранное вместо «Связаться»**:
  - Убраны кнопка «Связаться» и админ-подсказка про `MADMASK_CONTACT_EMAIL` / `madmask.contact_email`.
  - Добавлены:
    - степпер количества (− / текущее / +) через `cart#update`;
    - кнопка «В корзину» (добавляет +1) через `cart#add`;
    - кнопка «В избранное» (toggle) через `wishlist#toggle`;
    - кнопка «Убрать» из корзины (видна только когда qty > 0) через `cart#remove`.
  - Файл: `app/views/products/show.html.erb`.

## Риски / заметки

- Inline-script добавления `.js` выполнится рано, но при сверхстрогой CSP без `unsafe-inline` может быть заблокирован. Если CSP ужесточим — перенесём установку класса в разрешённый источник.
