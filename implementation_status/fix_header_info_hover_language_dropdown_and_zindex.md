## Контекст

В шапке сайта были замечания по UI:

- пункт меню «Инфо» открывался по клику (через `<details>/<summary>`), ожидалось открытие при наведении;
- выпадающее меню «Инфо» перекрывалось нижней горизонтальной линией шапки;
- переключение языков ожидалось в виде выпадающего списка с флагом и кодом языка.

## Что сделано

- Заменён блок «Инфо» в `app/views/shared/_header.html.erb` на паттерн “кнопка + панель” и добавлен Stimulus-контроллер `dropdown_controller.js`, который:
  - открывает меню при наведении и при фокусе,
  - закрывает при уходе курсора (с небольшой задержкой), клике вне и по `Esc`,
  - поддерживает клик по кнопке как fallback.
- Для выпадающих панелей добавлен высокий `z-index` (`z-[60]`), чтобы они гарантированно рисовались поверх декоративных элементов шапки.
- Нижняя градиентная линия шапки получила `pointer-events-none` и `z-0`, чтобы не перекрывать интерактивные элементы.
- Переключение языков (desktop) заменено на дропдаун с флагом и кодом языка (RU/EN).
