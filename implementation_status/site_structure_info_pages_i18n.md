# Site structure: info pages (RU-only)

## Статус

Документ обновлён после удаления мультиязычности.

- Локаль в URL больше не используется.
- Публичные инфо-страницы доступны по плоским маршрутам без префикса.
- Для старых адресов `/ru/...` настроены постоянные редиректы на новые URL.

## Актуальные маршруты инфо-страниц

- `/about`
- `/delivery-and-payment`
- `/lead-times-and-custom`
- `/materials-and-care`
- `/faq`
- `/contacts`

## Комментарий по истории

Старый этап с `/:locale` (`/ru/...`, `/en/...`) и language switcher заархивирован и не отражает текущее поведение приложения.
