## Улучшение отзывчивости управления 3D превью (OrbitControls)

### Цель
- Уменьшить ощущаемую «вязкость» при старте вращения модели мышью.

### Что изменено
- **`app/javascript/controllers/webgl_preview_controller.js`**
  - Увеличен `controls.dampingFactor` (быстрее «схватывает», меньше резинового ощущения).
  - Для desktop задан `rotateSpeed = 4.8` (ещё x2) и более быстрый `zoomSpeed`.
  - Добавлен **slow-drag boost** для desktop: при очень маленьких pointer-дельтах временно повышается `rotateSpeed`, чтобы чувствительность не «терялась» при медленном вращении.
  - Touch-профиль оставлен отдельным (как и раньше), изменения для touch не затрагивают desktop.

### Как проверить
- Открыть товар с 3D (`products#show`) и сразу потянуть мышью по модели.
- Субъективно: модель должна начинать вращаться заметно «живее» и без задержки ощущения.
