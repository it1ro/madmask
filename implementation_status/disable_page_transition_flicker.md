## Цель

Убрать раздражающее "моргание" при переходах по ссылкам (Turbo Drive навигация).

## Причина

Был включён эффект fade-in для `<main>`:

- `.page-main` имел `opacity: 0` по умолчанию в `app/assets/tailwind/application.css`
- Stimulus-контроллер `page-transition` на каждом `turbo:load` снимал/добавлял класс видимости, из-за чего контент успевал погаснуть и создавалось ощущение мигания.

## Решение (самый простой вариант)

- Отключили `data-controller="page-transition"` на `<main>` в `app/views/layouts/application.html.erb`.
- Убрали CSS, который скрывал `.page-main` по умолчанию, в `app/assets/tailwind/application.css`.
- Убрали класс `page-main` с `<main>`, чтобы даже при закэшированном старом CSS контент не мог остаться с `opacity: 0`.

## Ожидаемый результат

Переходы по ссылкам больше не вызывают кратковременное исчезновение основного контента.
